# Banco de Semillas del Invernadero

Sitio estático listo para GitHub Pages.

## Qué incluye
- Catálogo visual de semillas con imágenes del sobre e información posterior.
- Filtros por nombre y categoría.
- Bitácora de observación guardada en `localStorage`.
- Exportación de registros a CSV.
- Controles inclusivos: tema claro/oscuro, tamaño de fuente y lectura por voz.
- PWA básica con service worker.

## Cómo publicarlo en GitHub
1. Crear un repositorio nuevo.
2. Subir todos los archivos y carpetas tal como están.
3. Ir a **Settings > Pages**.
4. Activar publicación desde la rama principal (`main`) y carpeta `/root`.
5. Esperar el enlace público.

## Nota
Los registros de la bitácora se guardan localmente en el navegador. Si luego quieres, se puede conectar a Google Sheets o Supabase para guardar datos en línea.
