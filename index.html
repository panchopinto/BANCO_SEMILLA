Plantilla de investigación

Preguntas sugeridas

  • ¿Qué semilla germina más rápido en el invernadero?
  • ¿Cómo influye la frecuencia de riego en el crecimiento?
  • ¿Qué especie responde mejor al cultivo hidropónico?
  • ¿Qué diferencias aparecen entre sombra parcial y pleno sol?
  • ¿Cómo cambia el desarrollo según el tipo de sustrato?

Variables recomendadas

  • Variable independiente: riego, luz, sustrato, temperatura.
  • Variable dependiente: altura, hojas, color, floración, rendimiento.
  • Variables controladas: recipiente, fecha de siembra, cantidad de semillas.

Indicadores por etapa

  • Germinación: porcentaje y días de emergencia.
  • Crecimiento: altura semanal y número de hojas.
  • Sanidad: plagas, hongos, marchitez, clorosis.
  • Producción: masa, número de frutos, tamaño comercial.

Uso didáctico

Guía rápida para docentes

Qué permite este sitio

Organizar el banco de semillas, mostrar fichas visuales y registrar evidencias del proceso experimental de cada curso.

Cómo trabajar con alumnos

Formar equipos, asignar una especie, definir hipótesis, realizar observaciones periódicas y comparar resultados entre tratamientos.

Cómo publicar en GitHub

Sube todos estos archivos a un repositorio y activa GitHub Pages desde la rama principal.
